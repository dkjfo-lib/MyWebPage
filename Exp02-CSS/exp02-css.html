CSS SELECTORS

DECLEARING SELECTORS

  • use <elementName> to target specific element of the HTML
  • use <.className> to target specific class of the HTML
  • use <#className> to target specific id of the HTML

COMBINING SELECTORS

  • use <selector1, selector2> to apply rule to both selector1 and selector2
  • use <selector.className> to apply rule to elemnt whitch is both a selector1 as well as className
  • use <selector1 > selector2> to apply rule to selector2 whitch is a direct child of the selector2
  • use <selector1 selector2> to apply rule to selector2 whitch is any child of the selector2

PSEUDO-CLASS SELECTORS

  • use <selector:pesudo-className> to apply rules based on properties of the element

SUBLIME COMMANDS

  • use "alt+f3" select all matches of selection in the document
  • use "ctrl+shift+P" + "reindent" to reaply formating
